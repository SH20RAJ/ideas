> Not just an assistant.

## 5. 🏗️ Technical Architecture

### Core Loop
```text
User Input
→ Memory Retrieval
→ Context Injection
→ LLM Response
→ Memory Extraction
→ Storage
```

---

### Stack

#### Frontend
* Next.js (Vercel Chatbot fork)

#### Backend
* Node.js API routes

#### Model Layer
* Primary → glm-4.7-flash
* Advanced → nemotron-3-120b

#### Memory Layer
* Custom memory pipeline (MVP)
* Future → Memobase + Mem0 hybrid

#### Vector DB
* Supabase pgvector / Qdrant

#### Embeddings
* embeddinggemma-300m
